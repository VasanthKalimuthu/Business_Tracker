Sync Troubleshooting

1. API Keys: Open the code and replace "PASTE_YOUR_KEY" with your actual Firebase API Key.

2. Security Rules: In Firebase Console, go to Realtime Database > Rules and ensure they are set to: { ".read": true, ".write": true }.

3. Database URL: Check if your URL looks like https://your-app.firebasedatabase.app. Ensure it matches exactly.
