# Business Splitter App

A small static demo app to split a total amount equally between participants.

How to run:

1. Install dependencies:

```powershell
npm install
```

2. Start the lightweight server (will serve `public/` on port 3000):

```powershell
npm start
```

Open http://localhost:3000 in your browser.

Notes:
- This is a static front-end demo. For a real-time multi-user app you'd add a backend and websockets.
